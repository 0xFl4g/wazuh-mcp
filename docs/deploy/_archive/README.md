# Archived per-milestone deploy notes

These files preserve the deploy guide as it existed at each milestone tag (M2 through M5a). They remain for git-history archeology and for users diffing v0.x.y -> v1.0.0 deployments.

**For current deployment docs, read [../README.md](../README.md).**

Files in this directory may reference superseded surfaces (e.g., `m4b-writes.md` describes the M4b registration-time `write_allowlist` before M4c added the per-tenant resolver). They are not maintained.

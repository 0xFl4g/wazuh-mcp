# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be respectful and constructive in issues, PRs, and discussions.
- Focus feedback on the code, not the person.
- Assume good faith; ask questions before assigning blame.
- Harassment, discrimination, or personal attacks are not tolerated.

## Reporting

Report unacceptable behavior via GitHub's private vulnerability reporting (see [`SECURITY.md`](SECURITY.md)) or by emailing the maintainer at the address shown on their GitHub profile. Reports are handled confidentially and in line with the Contributor Covenant's enforcement guidelines.
